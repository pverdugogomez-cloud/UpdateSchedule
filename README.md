# PozasControl - Digital Twin Salar de Atacama

Este es un sistema de control de avances para faenas de construcción, diseñado específicamente para el monitoreo de pozas y gestión de equipos en el Salar de Atacama.

## Características

- **Carga de Avances**: Interfaz táctil y amigable para supervisores en terreno.
- **Dashboard**: Panel de control profesional con filtros por fecha, área y actividad.
- **Base de Datos en la Nube**: Integración total con Google Sheets para sincronización multiusuario.
- **Gestión de Usuarios**: Control de acceso granular (Administrador, Programador, Supervisor).
- **Gestión de Proyectos**: Aislamiento total de datos por obra/contrato.

## Configuración de Base de Datos

El sistema utiliza Google Sheets a través de una "Web App" de Google Apps Script. Para que funcione, el archivo `index.html` debe contener la URL `SHEETS_URL` válida.

## Despliegue

Este proyecto está diseñado para ejecutarse directamente desde GitHub Pages.
